### Como Instalar e Testar a Extensão

Siga os passos abaixo para instalar e testar a extensão em seu navegador Google Chrome:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/ArieviloDavidson/chromeExtension.git
    cd chromeExtension
    ```

2.  **Abra o Google Chrome:**
    Abra o navegador e navegue até `chrome://extensions/`.

3.  **Habilite o Modo de Desenvolvedor:**
    No canto superior direito da página de extensões, ative o "Modo de desenvolvedor".

4.  **Carregue a Extensão:**
    Clique no botão "Carregar sem compactação/Carregar expandida" que aparecerá no canto superior esquerdo.

5.  **Selecione a Pasta do Projeto:**
    Navegue até a pasta `chromeExtension` (a mesma que você clonou) e selecione-a.

6.  **Verifique a Instalação:**
    A extensão deve aparecer na sua lista de extensões e estar pronta para uso. Verifique se o ícone da extensão aparece na barra de ferramentas do Chrome.

Agora a extensão está instalada e pronta para ser testada!
